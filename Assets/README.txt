Q: Can I use these assets in a commercial project?
A: Please don't use anything but the code and standard assets in a released project unless otherwise specified.

Q: Why is my frame rate so low?
A: Please try adjusting your quality settings via Edit->Project Settings->Quality. From this menu, change the default quality level by selecting the arrow in the row labeled Default in the column of the platform you are building for.  When the arrow is selected, you will be presented with a drop down menu.

Q: Why are the polygons stretching?
A: Please try disabling static batching in Edit->Project Settings->Player.
